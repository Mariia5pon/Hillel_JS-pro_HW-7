7.1. Locking 7.2. Production through currying 7.3. Function with a cycle

7.1. Locking

Write a function that accepts 1 parameter. But it does not matter what was transmitted the first time, etc. All this with closures, for:
console.log(sum(4)); // 4
console.log(sum(6)); // 10
console.log(sum(10)); // 20
console.log(sum(7)); // 27

7.2. Production through currying

Create a function to calculate the product of two numbers, called as follows: name(5)(2) . The function must return a result (there must not be a log console in the middle of the function!)

7.3. Function with a cycle

The loop at each iteration prompts you to enter a number greater than 100 (but a maximum of 10 iterations of the loop). If the visitor entered a number less than one hundred - ask to enter again, and so on. If the user enters more than a hundred, the text or the loop completes all iterations, the function outputs the last user input to the console and terminates the function.
